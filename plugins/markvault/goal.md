El objetivo de este plugin es el de obtener un script que extraiga el texto de 
un PDF a un md. Aunque se pueda invocar mediante una skill, el contenido del 
pdf JAMAS (guardarail estricto) debe viajar fuera del equipo (ej: a servidores 
de Anthropic) sin consentimiento expreso o anonimizacion.
Otro objetivo es que la accion sea determinista y con gasto de tokens 0 (solo
la invocacion del comando por la skill)
