# Security Policy

## Supported versions

This repo is a monorepo of independently versioned plugins (each with its
own `version` in `.claude-plugin/plugin.json`). Security fixes are applied
to the latest version of each plugin on `main`; there are no long-term
support branches.

## Reporting a vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report suspected vulnerabilities privately using
[GitHub Security Advisories](https://github.com/dmarchena/agent-plugins/security/advisories/new)
for this repository. If you're unable to use that channel, contact the repo
owner, [@dmarchena](https://github.com/dmarchena), directly.

Please include:

- The affected plugin(s) and version(s).
- Steps to reproduce, or a minimal proof of concept.
- The potential impact as you understand it.

We'll acknowledge your report, investigate, and coordinate a fix and
disclosure timeline with you before any public write-up.
